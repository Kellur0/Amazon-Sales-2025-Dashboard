# Amazon Sales 2025 Dashboard

## Project Overview

This project presents an interactive Tableau dashboard that analyzes Amazon's 2025 sales performance using sales transaction data. The dashboard provides key business insights through interactive visualizations, enabling users to explore sales trends, product performance, payment methods, and regional sales distribution.

## Data Source

* **Dataset:** Amazon Sales Data 2025 (CSV)
* **Source:** Kaggle Amazon Sales Dataset: https://www.kaggle.com/datasets/zahidmughal2343/amazon-sales-2025/data
* **Visualization Tool:** Tableau Public: https://public.tableau.com/views/AmazonSales2025_17800865796840/Dashboard1?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

## Key Performance Indicators (KPIs)

* **Total Revenue:** Displays the total sales revenue generated.
* **Total Number of Customers:** Shows the total distinct customers in the dataset.

## Dashboard Visualizations

* Sales by Category (Bar Chart)
* Sales by Product (Horizontal Bar Chart)
* Monthly Sales Trend (Line Chart)
* Payment Methods Distribution (Pie Chart)
* Regional Sales Map (Geographic Map)

## Interactive Features

* Month filter for analyzing sales across different months.
* Payment Method filter to compare customer payment preferences.
* Interactive dashboard allowing users to explore multiple sales dimensions.

## User Stories

1. As a business manager, I want to monitor total revenue so I can evaluate overall sales performance.
2. As a sales analyst, I want to compare sales across product categories to identify the highest-performing categories.
3. As a marketing manager, I want to analyze monthly sales trends to understand seasonal demand and sales patterns.
4. As an operations manager, I want to visualize regional sales distribution to identify high-performing geographic locations.
5. As a finance analyst, I want to examine payment method usage to better understand customer purchasing preferences.

## Business Value

This dashboard helps decision-makers quickly identify sales trends, top-performing products, customer behavior, and regional performance through interactive visualizations. The insights can support data-driven decisions for marketing, inventory planning, and sales strategy.
